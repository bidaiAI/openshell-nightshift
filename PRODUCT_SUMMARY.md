# OpenShell NightShift — Product Summary

## What it is

OpenShell NightShift is a privacy-first task marketplace for operators and workers.

It is currently shipping as a **small-scale runnable beta** with:

- private and selective task visibility
- sealed bids
- controlled worker assignment
- receipt-based delivery
- selective reveal

## What is live today

- Web: [https://openshell-nightshift.vercel.app](https://openshell-nightshift.vercel.app)
- Backend health: [https://backend-preview-production.up.railway.app/health](https://backend-preview-production.up.railway.app/health)

## Current deployment model

- Vercel-hosted web control plane
- Railway-hosted API and reference worker
- isolated workspaces for safe concurrent beta sessions

## Product promise

NightShift helps teams delegate work without making everything public by default.

Its core value is:

- task privacy
- worker accountability
- verifiable delivery artifacts
- constrained evidence reveal when disputes occur

## Honest boundary

NightShift is deployable and runnable today, but it is still a controlled beta.

It is not yet:

- a fully decentralized worker network
- a final production auth system
- a final on-chain settlement system
